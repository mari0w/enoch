# skill-index

用于在用户询问“有哪些技能”时，列出当前项目内技能目录。

## 输出格式
- 列出技能名称 + 一句话用途
- 不包含全局技能
- 按分类分组（系统、自动化、文档、数据、规范）

## 当前技能（示例）
- system: 系统识别
- assistant-playbook: 解决问题流程
- tooling-installer: 安装策略
- os-automation-mac: macOS 自动化
- os-automation-linux: Linux 自动化
- doc-ops: 文档处理
- data-ops: 数据处理
- memory: 项目记忆读写
- codex-cli: Codex CLI 非交互规范
- codex-skills: 项目技能加载规范
- project-guidelines: 项目改造规范
